Overview

Project overview

Point2Skill explores how robots can learn goal-conditioned manipulation skills from interactive 3D inputs. The project focuses on connecting perception, action and learned motion primitives for flexible robot control.

Motivation

Why it matters

Many robotic systems struggle to generalize manipulation skills to new objects, goals and environments. This project investigates methods that make robot behavior more adaptable and easier to specify through spatial interaction.

Approach

Technical approach

The project combines 3D perception, goal-conditioned learning and diffusion primitives to generate robot actions that can adapt to different task configurations.

Status

Current status

This is an active research project. More information, publications, videos and code will be added as the project evolves.
