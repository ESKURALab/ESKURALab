Point2Skill

Interactive 3D Goal-Conditioned Diffusion Primitives for Robot Control.

Point2Skill investigates how robots can use interactive 3D target points to execute reusable manipulation skills such as picking, placing, inserting, opening and closing.

  • Robot learning
  • 3D perception
  • Diffusion policies
  • Manipulation
Visit project website ↗ Paper

Overview

What is Point2Skill?

Point2Skill is a framework for specifying reusable robot manipulation skills through 3D target points. Instead of relying only on language commands, the system grounds the intended interaction directly in the geometry of the scene.

Motivation

Why 3D points?

Language is useful for describing tasks, but it often lacks precise geometric grounding. By conditioning manipulation skills on 3D points, the robot can better understand where to interact and how to adapt the same skill to new objects or environments.

Method

How the framework works

The project combines interactive 3D goal selection with goal-conditioned diffusion primitives for robot control.

01

Select a 3D goal

A human or vision-language model selects a target point in the scene.

02

Condition the skill

The selected point conditions a reusable manipulation primitive.

03

Execute action

The robot executes skills such as pick, place, insert, open or close.

04

Compose tasks

Simple primitives can be sequenced to solve longer-horizon tasks.

Primitive skills

Reusable manipulation primitives

Point2Skill represents manipulation as a library of goal-conditioned primitives that can be reused across objects, categories and task configurations.

Pick

Grasp the selected object from a 3D target point.

Place

Move an object to a goal-specified target region.

Insert

Align and insert an object into a selected container or location.

Open / Close

Interact with articulated objects such as drawers or doors.
