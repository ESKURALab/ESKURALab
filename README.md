# ESKURA Lab Website

Starter static website for **ESKURA Lab** — *Embodied Skills and Knowledge for Unified Robot Adaptation*.

## Structure

```text
.
├── index.html
├── assets/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── main.js
│   └── img/
│       └── eskura-logo.jpeg
├── .nojekyll
└── README.md
```

## Edit these first

1. In `index.html`, replace `your-email@your-university.edu` with the lab email.
2. Replace `Your University, Your City` with the real affiliation and location.
3. Replace `github.com/your-organization` with your GitHub organization URL.
4. Replace the placeholder people and project cards with the real lab content.

## Publish with GitHub Pages

Recommended repository name for an organization website:

```text
<organization-name>.github.io
```

Then push these files to the `main` branch and enable GitHub Pages from repository settings.
