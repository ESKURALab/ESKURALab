# ESKURA Lab Website

Starter static website for **ESKURA Lab** — *Embodied Skills and Knowledge for Unified Robot Adaptation*.

## Structure

```text
.
├── index.html
├── assets/
│   ├── css/
│   │   └── styles.css
│   ├── js/
│   │   └── main.js
│   └── img/
│       └── eskura-logo.jpeg
├── .nojekyll
└── README.md
```


## Publish with GitHub Pages

Recommended repository name for an organization website:

```text
<organization-name>.github.io
```

Then push these files to the `main` branch and enable GitHub Pages from repository settings.
